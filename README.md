# Heart of the void

Jeu Java court inspiré du principe de "Battlescats" : placez des unités sur une grille,
composez des combinaisons et regardez-les s'affronter automatiquement.

Mécaniques principales : placement stratégique, types d'unités variés et progression des unités.
sur une grille de bataille pour occuper le territoire adverse et défendre votre cœur du vide.

## Description

Heart of the void est un projet de jeu tactique en temps réel / tour par tour (selon l'implémentation)
qui reprend les mécaniques de placement d'unités et de combat auto-battler popularisées par des jeux comme Battlescats.
Le projet cible une implémentation complète en Java afin d'être portable et facile à exécuter sur plusieurs plateformes.

## Principales fonctionnalités

- Placement stratégique d'unités sur une grille.
- Différents types d'unités avec comportements et statistiques propres.
- Système d'évolution / amélioration des unités.
- Modes solo et (prévu) multijoueur local ou en ligne.
- Interface graphique (Swing / JavaFX / libGDX selon l'implémentation choisie).

## Technologie

- Langage : Java (version recommandée : 17+)
- Build : Maven ou Gradle conseillés, sinon compilation manuelle via `javac`.
